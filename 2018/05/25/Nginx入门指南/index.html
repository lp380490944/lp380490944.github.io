Nginx入门指南

2018-05-25 22:43:28

基本操作指令

  • nginx -t 测试配置是否正确
  • nginx -s reload 加载最新配置
  • nginx -s stop 立即停止
  • nginx -s quit 优雅停止（完成当前任务后再退出）
  • nginx -s reopen 重新打开日志
